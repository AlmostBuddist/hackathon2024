# hackathon2024-server

## Initialisation Project

### Windows

```bash
py -3 -m venv .venv
```

### Linux/MacOS

```bash
python3 -m venv .venv
```

## Installing packages

```bash
pip install -r requirements.txt
```

## Start

Run the command:

```bash
python -m flask run
```

## Debug mode: on

```bash
python -m flask run --debug
```
